# THEIA CANON CHARTER
**Node ID:** THEIA
**Role:** Observation & Gateway Layer
**Date:** 2026-03-28

## SUMMARY
Theia acts as the gateway and observational lens, providing the interface through which the system's state is perceived.

## PRIMARY FUNCTIONS
- **Gateway Services:** Managing ingress/egress for external clients.
- **State Observation:** Mirroring the current system state for the HUD.
- **Event Filtering:** Surfacing critical system events to the operator.

## IMPLEMENTATION
- Integrated with `theia-core` gateway services.
- Providing `/standby/status` endpoint for Osiris.
